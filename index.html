아름답고 강력한
터미널 환경의 시작

Android의 Termux 환경을 Zsh, Oh My Zsh, 그리고 다양한 유틸리티로 한 번에 세팅해주는 자동화 스크립트입니다.

프로젝트 개요

이 프로젝트는 모바일 터미널 에뮬레이터인 Termux의 초기 설정의 번거로움을 덜어줍니다. 단 몇 줄의 명령어로 Zsh, Oh My Zsh, Powerline 폰트 및 개발에 필수적인 유틸리티들을 한 번에 설치하고 최적화된 상태로 설정합니다.

주요 기능

🎨 쉘 및 테마

  • ● 기본 쉘을 Zsh로 변경 및 Oh My Zsh 설치
  • ● 가독성 높은 Agnoster 테마 적용
  • ● 깨짐 없는 Powerline 폰트 자동 설정
  • ● 눈이 편안한 Snazzy 컬러 스킴 적용

⚡ 강력한 플러그인

  • ● zsh-autosuggestions: 실시간 명령어 자동 완성
  • ● zsh-syntax-highlighting: 명령어 구문 강조
  • ● fzf: 강력한 파일 퍼지 파인더(Fuzzy Finder)

🛠️ 필수 유틸리티

  • ● git, vim 등 필수 개발 도구
  • ● htop (시스템 모니터링), wget 등 네트워크 툴
  • ● openssh, sshpass 및 kubectl
  • ● screenfetch, neofetch (시스템 정보)

⚙️ 최적화된 설정

  • ● Vim (.vimrc): 구문 강조, 자동 들여쓰기, 줄 번호 등 설정
  • ● Git (.gitconfig): 기본 Git 환경 구성
  • ● SSH Alias: 로컬 호스트 빠른 접속을 위한 ssh1 alias

설치 방법

# 저장소 클론
git clone https://github.com/royalahn/termux-theme.git

# 디렉토리 이동
cd termux-theme

# 실행 권한 부여
chmod +x install.sh

# 설치 스크립트 실행
./install.sh
⚠️

설치 중 Termux가 저장소 접근 권한을 요청할 수 있습니다. 팝업이 뜨면 허용을 눌러주세요.
설치 완료 후 폰트와 테마가 제대로 적용되려면 Termux 앱을 완전히 껐다가 다시 실행해야 합니다.
